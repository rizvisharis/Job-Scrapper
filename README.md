# Web Scraping Tool 
## Run Locally

Clone the project

```bash
  git clone https://github.com/rizvisharis/Job-Scrapper.git
```

Go to the project directory

```bash
  cd Job-Scrapper
```

Install dependencies

```bash
  pip install
```

Start the server

```bash
  python flask_try.py
```

  
## Features
Extracting Job post from
- Stackoverflow Jobs
- Indeed Jobs
- easily get CSV formats

  
## Tech Stack

**Language:** Python, Html

**Library:** BeautifulSoup
